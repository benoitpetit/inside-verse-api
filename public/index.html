Inside Job API Documentation

Total API entries: 121

Seasons (2 entries)
/seasons Get all seasons
/seasons/1 Get a specific season by id
/seasons?_embed=episodes Get seasons with their related episodes
Episodes (18 entries)
/episodes Get all episodes
/episodes/1 Get a specific episode by id
/episodes?conspiracy_ids_like=14 Get episodes related to a specific conspiracy
/episodes?character_ids_like=1 Get episodes featuring a specific character
/episodes?season=1 Get episodes from a specific season
Quotes (44 entries)
/quotes Get all quotes
/quotes/1 Get a specific quote by id
/quotes?episode_id=1 Get quotes from a specific episode
/quotes?character_id=1 Get quotes by a specific character
/quotes/random Get a random quote
Characters (10 entries)
/characters Get all characters
/characters/1 Get a specific character by id
/characters?conspiracy_ids_like=1 Get characters involved in a specific conspiracy
/characters?_embed=quotes Get characters and their related quotes
Organizations (7 entries)
/organizations Get all organizations
/organizations/1 Get a specific organization by id
/organizations?related_conspiracy_ids_like=1 Get organizations related to a specific conspiracy
Locations (12 entries)
/locations Get all locations
/locations/1 Get a specific location by id
/locations?first_appearance_episode_id=1 Get locations featured in a specific episode
Technologies (12 entries)
/technologies Get all technologies
/technologies/1 Get a specific technology by id
/technologies?creator_like=Reagan%20Ridley Get technologies created by a specific character
Conspiracies (16 entries)
/conspiracies Get all conspiracies
/conspiracies/1 Get a specific conspiracy by id
/conspiracies?related_organization_ids_like=3 Get conspiracies related to a specific organization
