API Documentation

Episodes
Get all episodes /episodes
Get a specific episode by id /episodes/1
Get episodes related to a specific conspiracy /episodes?conspiracy_ids_like=14
Get episodes featuring a specific character /episodes?character_ids_like=1
Get episodes from a specific season /episodes?season=1
Quotes
Get all quotes /quotes
Get a specific quote by id /quotes/1
Get quotes from a specific episode /quotes?episode_id=1
Get quotes by a specific character /quotes?character_id=1
Get a random quote /quotes/random
Characters
Get all characters /characters
Get a specific character by id /characters/1
Get characters involved in a specific conspiracy /characters?conspiracy_ids_like=1
Get characters and their related quotes /characters?_embed=quotes
Organizations
Get all organizations /organizations
Get a specific organization by id /organizations/1
Get organizations related to a specific conspiracy /organizations?related_conspiracy_ids_like=1
Locations
Get all locations /locations
Get a specific location by id /locations/1
Get locations featured in a specific episode /locations?first_appearance_episode_id=1
Technologies
Get all technologies /technologies
Get a specific technology by id /technologies/1
Get technologies created by a specific character /technologies?creator_like=Reagan%20Ridley
Conspiracies
Get all conspiracies /conspiracies
Get a specific conspiracy by id /conspiracies/1
Get conspiracies related to a specific organization /conspiracies?related_organization_ids_like=3
